# Crypto Price Email Monitor with n8n

اتوماسیون ساده اما کاربردی برای پایش قیمت لحظه‌ای ارزهای دیجیتال مهم و ارسال گزارش دوره‌ای به ایمیل.

## هدف پروژه
دریافت قیمت به‌روز ارزهای دیجیتال (بیت‌کوین، اتریوم، تتر و ...) از API عمومی هر ۱ ساعت و ارسال خلاصه‌ای خوانا به ایمیل شخصی.

## فناوری‌ها و ابزارها
- n8n (نسخه self-hosted یا cloud)
- CoinGecko API (رایگان، بدون نیاز به کلید)
- Gmail node (یا SMTP دلخواه)
- Schedule Trigger

## ویژگی‌های اصلی
- اجرای خودکار هر ۶۰ دقیقه
- دریافت قیمت در دلار و تومان (اختیاری)
- نمایش درصد تغییر ۲۴ ساعته
- فرمت‌بندی خوانا برای ایمیل
- مدیریت خطا و لاگ‌گیری ساده

## نحوه استفاده / نصب مجدد
1. n8n را نصب کنید (docker / npm / cloud)
2. workflow.json را import کنید
3. Credentialهای Gmail را تنظیم کنید
4. Schedule را بر اساس نیاز تغییر دهید (Cron expression)

## گسترش‌های ممکن
- اضافه کردن شرط ارسال (فقط در صورت تغییر بیش از X%)
- ارسال به تلگرام / Slack به جای ایمیل
- ذخیره تاریخچه قیمت در Google Sheets یا Notion
- تحلیل ساده روند با AI

## نکات امنیتی
- هیچ‌گونه کلید API یا رمز عبور در کد قرار نگرفته است.
- از environment variables یا n8n credentials استفاده شود.
